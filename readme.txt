== Description ==

Плагин для Woocommerce. Добавляет поле для каждого товара, которое позволяет установить минимальное количество товара, которое будет добавляться в корзину. Например, если вы установилте минимальное количество 10, то клиент добавит 10 единиц товара в корзину, далее он может добавить +1 и т.д.
